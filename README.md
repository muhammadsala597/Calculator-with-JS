Calculator

A responsive calculator built with HTML, CSS, and JavaScript as part of my web development learning journey.

Features

- Addition, subtraction, multiplication, and division
- Percentage (%) calculations
- Parentheses support
- Square root (√) function
- Positive/negative (±) toggle
- Clear (CE) button
- Digital-style calculator display
- Animated button press effects
- Responsive CSS Grid layout

Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (DOM Manipulation)

Future Improvements

- Implement the DEL (Backspace) button
- Add keyboard support
- Improve error handling for invalid expressions
- Display calculation history
- Add scientific functions (x², xʸ, sin, cos, tan, log)
